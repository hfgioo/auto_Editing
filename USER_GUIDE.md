# Auto Editing - 使用指南

## 快速开始

### 1. 安装应用

#### Windows
1. 下载 `Auto-Editing-Setup.exe`
2. 双击运行安装程序
3. 按照提示完成安装

#### macOS
1. 下载 `Auto-Editing.dmg`
2. 打开 DMG 文件
3. 将应用拖到 Applications 文件夹

### 2. 配置 API Key

首次使用需要配置 Google Gemini API Key：

1. 访问 [Google AI Studio](https://makersuite.google.com/app/apikey)
2. 登录你的 Google 账号
3. 点击 "Create API Key"
4. 复制生成的 API Key
5. 在应用中打开"设置"页面
6. 粘贴 API Key 并保存

### 3. 开始使用

#### 上传视频

**方式一：拖拽上传**
1. 打开"上传视频"页面
2. 将视频文件拖拽到上传区域
3. 支持同时上传多个文件

**方式二：选择文件**
1. 点击"选择文件"按钮
2. 在文件选择器中选择视频
3. 支持多选

**支持的格式**
- MP4
- MOV
- AVI
- MKV
- WebM

#### 处理视频

1. 上传视频后，点击"开始处理"
2. 应用会自动：
   - 使用 AI 分析视频内容
   - 识别精彩片段
   - 自动剪辑
   - 生成字幕
   - 添加背景音乐（如果启用）
3. 在"处理进度"页面查看实时进度

#### 下载结果

1. 处理完成后，在"处理进度"页面
2. 点击"下载视频"按钮
3. 或点击"预览"查看效果

## 功能详解

### AI 智能分析

应用使用 Google Gemini AI 分析视频：

- **内容识别**：识别视频中的场景、人物、动作
- **精彩片段**：自动标记重要时刻
- **评分系统**：为每个片段打分（0-100）
- **语音转文字**：自动生成字幕文本

### 自动剪辑

基于 AI 分析结果：

- 选择评分最高的片段（最多5个）
- 按时间顺序合并
- 保持流畅的过渡
- 自动调整时长

### 字幕生成

- 自动语音识别
- 生成 SRT 格式字幕
- 嵌入到视频中
- 支持自定义样式（开发中）

### 背景音乐

- AI 推荐音乐风格
- 自动音量调节
- 与原音混合
- 音乐库（开发中）

## 设置选项

### 视频质量

- **低 (720p)**：文件小，处理快
- **中 (1080p)**：平衡质量和大小
- **高 (1080p 高码率)**：推荐设置
- **超高 (4K)**：最佳质量，文件大

### 输出路径

- 默认：与原视频相同目录
- 自定义：点击"浏览"选择文件夹
- 每个视频会创建独立的项目文件夹

### 自动功能

- **自动生成字幕**：启用/禁用字幕生成
- **自动添加音乐**：启用/禁用背景音乐

## 常见问题

### API Key 相关

**Q: 如何获取 API Key？**
A: 访问 Google AI Studio，登录后创建新的 API Key。

**Q: API Key 是否收费？**
A: Google Gemini 提供免费额度，超出后按使用量计费。

**Q: API Key 安全吗？**
A: API Key 仅存储在本地，不会上传到任何服务器。

### 视频处理

**Q: 支持哪些视频格式？**
A: MP4, MOV, AVI, MKV, WebM 等常见格式。

**Q: 视频大小有限制吗？**
A: 建议单个文件不超过 2GB，时长不超过 30 分钟。

**Q: 处理需要多长时间？**
A: 取决于视频长度和质量设置，通常 5 分钟视频需要 2-5 分钟。

**Q: 可以批量处理吗？**
A: 可以，同时上传多个视频会依次处理。

### 错误处理

**Q: 提示"API Key 无效"？**
A: 检查 API Key 是否正确，是否有足够的配额。

**Q: 处理失败怎么办？**
A: 查看错误信息，常见原因：
- 视频格式不支持
- 磁盘空间不足
- 网络连接问题
- API 配额用尽

**Q: 应用崩溃了？**
A: 重启应用，检查系统资源（内存、磁盘）。

## 高级技巧

### 优化处理速度

1. 使用较低的视频质量设置
2. 关闭不需要的功能（字幕/音乐）
3. 确保有足够的磁盘空间
4. 关闭其他占用资源的程序

### 提高输出质量

1. 使用"高"或"超高"质量设置
2. 确保原视频质量良好
3. 使用稳定的网络连接（AI 分析需要）

### 自定义工作流

1. 先处理一个测试视频
2. 根据结果调整设置
3. 批量处理相似视频
