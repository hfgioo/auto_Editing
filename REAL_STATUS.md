## ⚠️ 限制和说明

### 1. Gemini API
**状态**: 演示模式  
**原因**: 
- 浏览器环境无法直接读取大文件
- 需要在 Electron 主进程中处理
- 当前返回模拟数据用于演示

**实际使用**:
- 需要配置真实的 Gemini API Key
- 需要实现主进程文件读取
- 需要处理 API 限制和错误

### 2. FFmpeg 视频处理
**状态**: 代码完整，需要 Electron 环境  
**原因**:
- FFmpeg 需要在 Node.js 环境运行
- 浏览器无法直接操作文件系统
- 需要通过 IPC 调用主进程

**实际使用**:
- 在 Electron 环境中可以正常工作
- 需要安装 FFmpeg 二进制文件
- 需要处理大文件和长时间处理

### 3. 文件上传

**功能**:
- ✅ 拖拽上传
