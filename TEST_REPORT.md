## 已知限制（浏览器模式）

### ⚠️ 不支持的功能
1. **文件系统访问**
   - 无法选择输出目录
   - 无法真实读取视频文件
   - 无法导出到本地文件系统

2. **视频处理**
   - 无法调用 ffmpeg
   - 无法真实分析视频
   - 无法调用 AI API

3. **数据库**
   - 使用 localStorage 代替 SQLite
   - 数据量有限制

### ✅ Electron 模式将支持
- 真实的文件系统访问
- ffmpeg 视频处理
- AI API 调用
- SQLite 数据库
- 系统通知
