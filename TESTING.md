## 测试用例

### 视频处理流程
1. 上传单个视频文件
2. AI 分析视频内容
3. 自动剪辑精彩片段
4. 生成字幕
5. 添加背景音乐
6. 导出最终视频

### 错误处理
- 无效文件格式
- API Key 错误
- 网络错误
- 磁盘空间不足
- FFmpeg 错误

## 测试数据

### 测试视频
- 短视频 (< 1分钟)
- 中等视频 (1-5分钟)
- 长视频 (> 5分钟)
- 不同格式 (MP4, MOV, AVI)
- 不同分辨率 (720p, 1080p, 4K)
